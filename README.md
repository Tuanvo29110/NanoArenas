# Nano-Arenas
A lightning fast Arena system for maps of any size with ultrafast resetting.

# Nano Arenas v2.0 Update

## Changes
  - [x] Fully rewrite Reset System
  - [x] Full Support for Resetting Liquid Physics & Interaction caused changes
  - [x] Full Support for Resetting Non-Player edits to arenas eg: through worldedit or any other server edits
  - [x] Full Support for Resetting Block & Entity Explosions
  - [x] 5 New Reset modes
  - [x] New Asynchronous Block Resetting (Bypassing Bukkit) (upto 7.9M blocks per second // 1ms reset for average match)
  - [x] Better FAWE Resetting
  - [x] New Schematic Resetting
  - [x] New & Improved Bukkit Resetting (~80K blocks per second)
  - [x] New destroyable Arena rule
  - [x] New In-Built WorldManager to remove need for multiverse (Toggleable) 
  - [x] Optimized Block Tracking
  - [x] Optimized Reset System

  - [x] Optimized Entity Explosion handler
  - [x] Arena Backup , save & Load system
